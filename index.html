📋 MOT History

Enter a UK registration to fetch the full MOT history via a server‑side proxy (Vercel).

Select all (Ctrl+A) and copy (Ctrl+C).

🛡️ Fallback (if the server fails)

If you see a "No MOT table found" error, the MOT site may have blocked the server. Use this guaranteed manual method instead:

  1. Copy the script below.
  2. Open the official MOT page for your reg.
  3. Paste the script into the address bar and press Go.
  4. A blue box will appear with all the data – just copy it.


javascript:(function(){let t=document.querySelector('table');if(!t)return alert('No MOT table found on this page.');let b=document.createElement('textarea');b.value=t.innerText;b.style.cssText='position:fixed;top:5%;left:5%;width:90%;height:80%;z-index:9999;font-size:18px;background:white;color:black;border:4px solid #1e293b;border-radius:8px;padding:12px;';document.body.appendChild(b);b.select();alert('✅ Text selected – tap Copy or long‑press the box.');})();
